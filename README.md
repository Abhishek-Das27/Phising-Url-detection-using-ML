# Phising-Url-detection-using-ML
* **Frontend:** Semantic HTML5, CSS Variables, Flexbox/Grid Layouts, and **Lucide Icons** via CDN.
* **Backend:** Python 3, Flask framework, Cross-Origin Resource Sharing (CORS) security layer, and Regular Expression text parsing filters.

Follow these steps to set up and run PhishShield locally on your machine.

Ensure you have Python 3.8+ installed on your system.

```bash
